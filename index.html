UX Burlington

Burlington, Vermont

An annual single day, conference-style gathering, focused on what it really takes to build a great user experience on the web.

The 2018 UX Burlington conference took place June 15, 2018. Sign up to the mailing list — hope you see you next year!

The theme for the 2018 confernce was The Path to Impactful UX. Each stand-alone presentation tracked the progression of a UX project.
